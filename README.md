## Run Locally

Clone the project

```bash
  git clone https://github.com/Saieeshsrao/Book-Store.git

Go to the backend
```bash 
cd backend
```
```bash
  npm i
  npm run dev
```
Go to the frontend
```bash
  cd frontend
```
```bash
npm i
```
```bash
npm run dev
```
